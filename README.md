# Smart AI Resume Analyzer

**Your Intelligent Career Partner**

Smart AI Resume Analyzer is an all-in-one tool to analyze, optimize, and craft resumes that stand out, helping you land your dream job.

**Developed by:** Kranthi Kumar  
**Academic Context:** Final Year B.Tech in Computer Science and Engineering Project at the Institute of Aeronautical Engineering College (April 2024 - May 2024)

---

## 🚀 Features

* **Deep Resume Analysis:** Includes an ATS Compatibility Score, Keyword Gap Analysis, Role-specific Feedback, and a Skills Gap Breakdown.
* **AI-Powered Resume Builder:** Offers smart content suggestions, ATS-optimized formatting, and fully customizable sections.
* **AI Optimization Engine:** Features keyword highlighting, content enhancement tips, and industry-specific insights.

## 💻 Tech Stack

* **Frontend & Backend:** Streamlit, Python
* **Modules:** spaCy, Python-docx, PyPDF2, scikit-learn, Plotly, NLTK
* **AI Integration:** Google Gemini API
* **Database:** SQLite3
